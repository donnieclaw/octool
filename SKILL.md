# OCTool Visual Interface
A visual tool to easily manage openclaw configuration, dotfiles, backups, and restores. It allows easy injection of terminal commands into your bash_profile and seamless sync.

## octool

一键打开可视化界面。终端或聊天框运行：

```bash
open {baseDir}/oc-tool.html
```
